# Game-dev-Journey-Decision-Companion-System
This is a decision companion system for your journey at game development.<br>
The main purpose of this project is to help the user to find the best path in their game development journey.<br>
This is a CLI program will include basic purpose like what game engine should they choose and what game to make.<br>
The user priority is taken in and is used to find the best path forward for the user.<br>
